In this lesson we will explain how to use FumbleChain's Blockchain explorer.

When running the FumbleChain daemon as explained in the challenges, the blockchain explorer is enabled by default. If you're running the daemon.py script directly, you need to pass the --explorer argument to enable it. By default the explorer will listen on port 20601 but this can be changed with the --explorer-port argument.

When running via fc-docker.sh as explained in challenges, the explorer port is the one specified with the -E option.

In summary, the full command to enable the explorer when running daemon.py directly is ./daemon.py --explorer start.

The Blockchain explorer is a great tool to visually explore blocks, transactions, blockchain parameters such as the magic and current target, as well as inspect the full transaction history for specific wallet addresses.

If you already ran the fc-docker.sh command from a challenge, then the daemon should be running and the blockchain explorer should be available at http://localhost:20601.

That concludes this lesson about the Blockchain explorer.
