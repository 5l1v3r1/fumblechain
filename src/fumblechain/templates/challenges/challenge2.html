Connecting to mainnet2

You can re-use the same client as the one used in Challenge 1. No need to re-download it if you already have it. To start the node and connect to mainnet2:

./fc-docker.sh -n fc-mainnet2 -i fc -p 2224 -P 1339 -E 20603 -H {{ CHALLENGE2_MAINNET2_NODE_URL }} --magic={{ CHALLENGE2_MAINNET2_MAGIC}}

To use the CLI client on mainnet2:

docker exec -it fc-mainnet2 ./cli.py --api-url http://localhost:1339

Once again, your mission is to buy the flag for challenge 2 here on the FumbleStore.
