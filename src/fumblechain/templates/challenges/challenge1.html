FumbleCorp inc. introduced its latest innovative blockchain-based product named FumbleChain. It is an infrastructure allowing anyone to securely transfer FumbleCoins, a digital currency.

The FumbleChain network (mainnet) is the production network and the one people use to exchange real funds. Developers can use the FumbleChainDev network as a test network (testnet) for building the future of FumbleChain.

Client download

Download the client here: fumblechain.tar.gz

Then extract the archive and change to the fumblechain directory:

tar xf fumblechain.tar.gz
cd fumblechain

Connecting to the mainnet

To start the node and connect to the mainnet:

./fc-docker.sh -p 2222 -n fc-mainnet -i fc -P 1337 -E 20601 -H {{ CHALLENGE1_MAINNET_NODE_URL }}

Connecting to the testnet

To start the node and connect to the testnet:

./fc-docker.sh -p 2223 -n fc-testnet -i fc -P 1338 -E 20602 -H {{ CHALLENGE1_TESTNET_NODE_URL }} --magic={{ CHALLENGE1_TESTNET_MAGIC }}

Free coins can be requested from the MoneyMaker app on the testnet.

Interacting with the blockchain

The script cli.py can be used to generate a wallet, make transactions and mine blocks. Make sure to specify the correct --api-url parameter depending on what you chose when running daemon.sh. For example, to connect to the daemon connected to the mainnet:

docker exec -it fc-mainnet ./cli.py --api-url http://localhost:1337

Or to connect to the testnet:

docker exec -it fc-testnet ./cli.py --api-url http://localhost:1338

Your mission is the get the flag which is for sale here on the FumbleStore. Note that the store expects real payments, that is from the mainnet.

Using the blockchain explorer

After you have started the daemon with one of the ./fc-docker.sh ... -E <explorer_port> ... commands, you may browse http://localhost:<explorer_port> to access to blockchain explorer. Also feel free to have a look at the lesson about this tool.
