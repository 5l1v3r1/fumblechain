Connecting to mainnet3

You can re-use the same client as the one used in Challenge 1. No need to re-download it if you already have it.

To start the node and connect to mainnet3:

./fc-docker.sh -n fc-mainnet3 -i fc -p 2225 -P 1340 -E 20604 -H {{ CHALLENGE3_MAINNET3_NODE_URL }} --magic={{ CHALLENGE3_MAINNET3_MAGIC}}

To use the CLI client on mainnet3:

docker exec -it fc-mainnet3 ./cli.py --api-url http://localhost:1340
