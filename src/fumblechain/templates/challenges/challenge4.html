For its 3rd anniversary, FumbleCorp inc. released an early prototype version of FumbleChain. Back then, FumbleCorp was very proud of its transaction system. They were pretty sure every transaction parameter was thouroughly checked, making the system unhackable.

This early version requires using a slightly different version of the FumbleChain client.

Client download

Download the client here: fumblechain-ch4.tar.gz

Then extract the archive and change to the fumblechain directory:

tar xf fumblechain-ch4.tar.gz
cd fumblechain

Connecting to mainnet4

To start the node and connect to mainnet4:

./fc-docker.sh -p 2225 -n fc-mainnet -i fc -P 1341 -E 20605 -H {{ CHALLENGE4_MAINNET4_NODE_URL }} --magic={{ CHALLENGE4_MAINNET4_MAGIC}}

Interacting with the blockchain

Use the blockchain explorer's embedded Web Wallet: Mainnet4 explorer
